修改文件读写权限
	
	chmod u+x 文件名  给文件加上可执行权限，然后在Python代码前加上：
	#!/usr/bin/python 这样以后就可以直接运行 ：
	./file.py
	sudo chmod 666 文件名   变成可读写
	
	
文件操作：
	
	mkdir folder
	touch file
	cp file folder/
	mv file folder/
	rm -r folder
	
	cat t.py > t1.py  把t.py东西复制到t1.py
	cat t.py t1.py >  t2.py  把前两个打包放到t2
	cat t3 >> t2.py 把t3内容加到t2后面
	
	ls -l 可以查看文件打权限
